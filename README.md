# Legacy.EXE-App

**Lifestyle RPG** — Where Cyber Meets Primal Empire

## About

Legacy.EXE is a lifestyle RPG that fuses dark fantasy world-building from the *Warriors of Primal Empire* series with cyberpunk mechanics. Become **The Uncompiled** — a digital warrior navigating a fractured world of ancient empires, neural augmentations, and legacy code that binds reality.

## World Lore

In the year of the Great Compilation, the Primal Empire's ancient runes clashed with emergent AI overlords. The world fragmented into **Legacy Realms** — hybrid zones where magic and machine coexist uneasily.

You are **The Uncompiled**, a rogue executable awakened from the void. Armed with:

- **Arm Compiler**: Forge weapons from raw code and primal essence.
- **Neural Spine Integration**: Channel forbidden spells through synaptic firewalls.
- **Velocity Striders**: Dash through digital ley lines and shadowed battlefields.

**Archetypes**:
- **Warrior**: Master health, fitness, and brutal combat.
- **Builder**: Shape productivity systems and forge legacies.
- **Scholar**: Unravel arcane algorithms and hidden knowledge.

Face undead hordes in epic battles, build your empire, and decide the fate of the Legacy.EXE.

> "The future of this series — and your legacy — hinges on you, the reader/player." — Raymond M. Williams

## Features

- Skill trees & progression
- Cybernetic augmentations
- Dark fantasy storytelling
- Mobile-first experience (Capacitor)
- Supabase backend for persistence

## Getting Started

1. Clone the repo: `git clone https://github.com/DAEXELabs/Legacy.EXE-App.git`
2. Install dependencies: `npm install`
3. Configure environment: Copy `.env.example` to `.env.local` and add your Supabase keys
4. Run locally: `npm run dev`

## Deployment

- Netlify (via `netlify.toml`)
- Android builds with Capacitor
